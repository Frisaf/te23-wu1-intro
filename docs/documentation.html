Errors I got and how I solved them

  • I didn't put the link to script.js in the
    Solved by putting the link to script.js in the head.
  • Contrast errors with Wave
    The colour of the links gave a contrast error, so I switched the looks of them with css by making them the same colour as the rest of the text, but bolded and of course underlined.
  • The paragraph element
    It can't be a child of a list element, so I fixed it with <br> instead.

Fel jag stötte på

  • Jag satte inte länken till script.js i dokumentets head
    Löstes genom att sätta script.js i dokumentets head.
