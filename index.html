HTML text elements

To format a text with HTML, one can use different heading elements. Paragraphs are used for paragraphs (obviously).

You can read more about HTML text elements at Mozilla

You can see a few examples of HTML text elements down below.

  • <h1> - The heading 1 element, the biggest heading.
  • <h2> - The heading 2 element, the second biggest heading.
  • <a> - Creates a hyperlink. The link itself is specified in the href attribute.
  • <div> - Creates a "box", which you can assign a CSS class to style it. Useful for structuring and designing the website.
